# liquidity_constraints
二期間モデルを用いて流動性規制の影響を受ける条件について考察しました。

## 概要
本リポジトリは、流動性制約（借入制約）を導入したマクロ経済学の二期間モデルをMATLABで数値計算し、可視化するプログラムです。

## 実行環境
* MATLAB

## ファイル構成
* `compare_constraints_w1_grid.m`: 若年期の富 $w_1$ を変化させ、流動性制約の有無による最適貯蓄 $a$ の変化を比較するスクリプトです。
* `compare_constraints_ratio_grid.m`: 利子率と時間選好率の比 $ratio = r / \rho$ を変化させ、流動性制約の有無による最適貯蓄 $a$ の変化を比較するスクリプトです。
* `compare_constraints_ratio_w2_surf.m`: $ratio$ と老年期の所得 $w_2$ の2変数を変化させ、最適貯蓄 $a$ の変化を3D曲面で比較するスクリプトです。
* `constraint_boundary_ratio_w2.m`: $ratio$ と $w_2$ が異なるケースにおいて、流動性制約が効いて貯蓄が0になる境界線を等高線でシミュレーションするスクリプトです。

## モデルの前提
* **効用関数**: CRRA型効用関数
* **予算制約式**:
  $$c_1 + a = w_1$$
  $$c_2 = (1+r)a + w_2$$
* **流動性制約**: $a \ge 0$ （負の貯蓄＝借入を許容しない）
* **オイラー方程式の残差関数**:
  $$R(a) = \beta(1+r) \left( \frac{w_1 - a}{(1+r)a + w_2} \right)^\gamma - 1$$

## プログラムのポイント
* `fzero` 関数と無名関数（`@(a)`）を用いて、残差関数 $R(a) = 0$ となる最適点（ゼロ点）を探索しています。
* 流動性制約下で $R(0) < 0$ （本来は借入をして消費を増やしたい状態）となる場合、最適貯蓄は $a = 0$ に張り付くように処理しています。
* `meshgrid` と `contour` を用いて、第2期の所得 $w_2$ と比率 $ratio = r/\rho$ の変化に対する制約の境界を等高線として可視化しています。

## 分析結果の要約
1. **二期目の所得 ($w_2$) の影響**: 一期目の所得 ($w_1$) に対して $w_2$ が大きいほど、将来の返済能力が高まるため借入インセンティブが働き、流動性制約の影響を受けやすくなります。
2. **利子率と時間選好率 ($r/\rho$) の影響**: 時間選好率 $\rho$ に対して利子率 $r$ が小さい（お金を使うコストよりも、今すぐお金を使いたい気持ちが勝る）場合、流動性制約の影響を受けやすくなります。
3. **依存度の比較**: 制約の有無は「 $r$ と $\rho$ の比」と「 $w_1$ と $w_2$ の比」の双方に依存するものの、特に所得の比 ($w_1, w_2$) の方に大きく依存するといえます。
